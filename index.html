Hello World! I am learning game development (specifically Love2D) using @josefnpat's excellent Love2D tutorial. This game is Battlefield 1942 modeled on the game of the same name on GameBoy Color. I am simply adding features to the original game in the tutorial.

Running the game

As of now, you need Love2D installed to play the game. I will publish a binary soon. Grab the assets (7z), unzip assets.zip to the cloned folder, then:

love main.lua
